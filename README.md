# beyzaekrem.github.io

Beyza Ekrem - Personal Portfolio Website

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Create a repository named `beyzaekrem.github.io` on GitHub.

2. Push the project:

```bash
git init
git add .
git commit -m "Initial commit: portfolio website"
git branch -M main
git remote add origin https://github.com/beyzaekrem/beyzaekrem.github.io.git
git push -u origin main
```

3. Deploy:

```bash
npm run deploy
```

4. In GitHub repo settings: Settings > Pages > Source: select `gh-pages` branch.

The site will be live at `https://beyzaekrem.github.io`
